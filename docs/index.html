// HOW IT WORKS

No cloud.
No compromise.

Every message goes directly device-to-device. Your data never touches a server.

01

BLE DISCOVERY

Your phone broadcasts a BLE beacon and scans simultaneously. Nearby STRANGER users appear on your radar within seconds — no WiFi required.

02

DIRECT CONNECT

Tap a stranger. They get an accept/reject prompt. Once accepted, a WiFi Direct tunnel forms — up to 250 Mbps, zero intermediaries.

03

KEY EXCHANGE

Both devices instantly generate X25519 keypairs, exchange public keys, and derive a shared AES-256-GCM secret via ECDH.

04

ENCRYPTED CHAT

Every message is AES-256-GCM encrypted before it leaves your device. When the chat ends, keys vanish and messages auto-wipe.

// SECURITY

Military-grade.
Zero knowledge.

We don't even have a server that could be hacked. Messages are encrypted before leaving your phone — and keys disappear when the chat ends.

Perfect forward secrecy means captured traffic today can't be decrypted tomorrow.

// Fresh keypair every session
const myKey = X25519.generate()
// Exchange over Nearby (public — safe)
send( myKey.publicKey )
receive( peer.publicKey )
 
// Shared secret — never transmitted
const secret = ECDH( myKey.private, peer.public )
const aesKey = HKDF_SHA256( secret )
 
// Encrypted with random nonce each time
const msg = AES_256_GCM.encrypt(
  plaintext, aesKey, nonce.random()
)
 
// Disconnect → keys gone forever
aesKey.destroy()
